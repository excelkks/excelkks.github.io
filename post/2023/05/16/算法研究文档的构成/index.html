算法研究文档的构成

2023-05-16

一篇优化算法研究过程的文档基本的构成包括以下几个部分:

  1. 简介. 简单介绍背景

  2. 策略.

    • 基本策略

    • 策略的合理性分析

    • 提出策略的预期

  3. 已有原始算法的数据分析

    • 统计数据的相关性

    • 通过已有数据评估预期

    • 总结

  4. 对优化算法的数据

  5. 总结优化算法
